1. Acceso al sistema

La plataforma usa rutas por idioma: /es para español y /en para inglés.

  • Catálogo: /es/courses
  • Login: /es/login
  • Panel personal: /es/dashboard
  • CMS académico: /es/dashboard/cms
  • Administración: /es/dashboard/admin

3. Roles

Administrador

Gestiona pagos, usuarios, roles, certificados, auditoría y cursos desde el CMS.

Instructor

Gestiona sus cursos, ediciones, módulos, sesiones, videos, materiales, evaluaciones, banco de preguntas y analítica.

Estudiante

Consulta el catálogo, se matricula, completa sesiones, realiza evaluaciones, recibe feedback y emite certificados si cumple criterios.

4. Catálogo y matrícula

  1. Entrar a /es/courses.
  2. Buscar o seleccionar un curso.
  3. Revisar ficha, requisitos, duración, competencias y certificado disponible.
  4. Seleccionar edición si aplica.
  5. Matricularse.

Los cursos gratuitos permiten acceso directo. Los cursos pagos pueden requerir pago y verificación administrativa.

5. Experiencia del estudiante

El estudiante ve módulos, sesiones, videos, materiales, actividades, progreso, evaluación y certificado.

Completar una sesión

  1. Abrir el curso.
  2. Revisar la sesión.
  3. Ver el video o recurso principal.
  4. Consultar materiales complementarios.
  5. Pulsar Completar.

El progreso se recalcula automáticamente. Solo cuentan sesiones publicadas.

6. Videos

Las sesiones pueden usar YouTube, Vimeo, enlace externo o video subido a la plataforma. Se recomienda usar plataformas externas cuando el video ya exista allí y reservar la subida local para contenido propio o cerrado.

7. Materiales complementarios

Cada sesión puede incluir PDF, lecturas, diapositivas, datasets, enlaces externos, objetos de aprendizaje en repositorios o archivos subidos.

8. Evaluaciones

La evaluación soporta selección múltiple, verdadero/falso y respuesta corta.

  • Nota mínima.
  • Límite de intentos.
  • Retroalimentación inmediata.
  • Aleatorización de preguntas y opciones.
  • Corrección automática del lado servidor.

Las respuestas correctas no se envían al navegador antes del envío.

9. Certificados

Para emitir certificado, el estudiante necesita matrícula activa, progreso 100%, curso con certificado habilitado, evaluación aprobada y no tener certificado previo para esa matrícula.

La verificación pública se consulta en /es/verify/{badgeId} o mediante JSON-LD en /api/verify/{badgeId}.

10. CMS académico

Ruta: /es/dashboard/cms.

Crear curso

  1. Entrar como administrador o instructor.
  2. Abrir el CMS académico.
  3. Pulsar Nuevo.
  4. Completar ficha, objetivos, público, requisitos, competencias, duración, nivel, idioma, precio y visibilidad.
  5. Guardar.

Publicar curso

El CMS exige ficha básica completa, al menos una edición publicada, al menos una sesión publicada con video y precio válido si el curso es pago.

11. Ediciones

Una edición representa una cohorte o versión de un curso. Permite definir fechas, cupo, edición por defecto y matrículas manuales.

12. Módulos

Los módulos organizan el contenido. Permiten agrupar sesiones y definir orden académico.

13. Sesiones

Desde la pestaña Sesiones se crean clases, se asocian a módulos, se agregan videos, duración, descripción, práctica y materiales complementarios. Las sesiones archivadas dejan de mostrarse a estudiantes.

14. Banco de preguntas

Ruta: CMS Académico > Banco.

Cada pregunta puede tener tipo, texto bilingüe, opciones, respuesta correcta, feedback, puntos, etiquetas, dificultad, tema y módulo asociado.

Guardar preguntas

  1. Crear preguntas en Evaluación.
  2. Ir a Banco.
  3. Pulsar Guardar evaluación como banco.

Usar preguntas del banco

  1. Ir a Banco.
  2. Definir filtros opcionales: cantidad, etiqueta, dificultad, módulo o tema.
  3. Pulsar Usar banco en evaluación.

15. Analítica MOOC

Ruta: CMS Académico > Analítica.

Muestra matrículas, progreso promedio, finalización, aprobación, certificados, intentos, mejor nota promedio y desglose por edición.

16. Panel de administración

Ruta: /es/dashboard/admin.

Permite gestionar pagos, instrucciones de pago, usuarios, certificados y auditoría.
